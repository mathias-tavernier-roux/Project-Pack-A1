Liste des Commandes Git Les Plus Importantes

La Base Pour Utiliser Git

Utilité de La Commande Commande Git Description Rapide
Creer Une Branche git branch newBranch Crée dans le repo local la nouvelle branche newBranch qui est un fork de la branche courante.
Renommer Une Branche git branch -m locBranch Renomme la branche courante avec le nouveau nom locBranch.
Supprimer Une Branche du Depot git branch -d locBranch Supprime la branche locBranch du repo local.
Traquer Une Branche Distance git branch -u origin/remBranch Ordonne à la branche courante de traquer la branche distante remBranch.
Lister Les Branches Sur Le Depot Local git branch -v Liste les branches sur le repo local.
Lister Les Branches Sur Le Depot Distant git branch -v -a Liste les branches sur le repo distant.
Compter Le Nombre de Branches Sur le Depot Distant git branch -r | wc -l Compte le nombre de branches effectivement présentes sur le repo distant.
Voir L'Historique des Commits Effectués git log --pretty=oneline Affiche l’historique des commits effectués sur la branche courante, chaque commit tenant uniquement sur une ligne.
Lister Les Commits Recents qui N'Ont Pas Etait Envoyé git log --name-status HEAD^..HEAD Liste tous les commits récents effectués, mais non encore pushés.
Créer un Tag Dans la Branche Courante git tag nomTag Crée un tag de nom nomTag sur la branche courante.
Donner les Detail d'Un Tag De La Branche Courante git show nomTag Donne les détails sur le tag nomTag de la branche courante.
Lister les Tag de La Branche Courante git tag Liste tous les tags placés sur la branche courante.
Remettre En Cache Les Modifications de La Branche Courante git stash Remet en cache (appelé Remise) les modifications sur la branche courante.
Lister Les Modification git stash list Liste tous les stash sur le repo local.
Voir Les Fichiers Modifié Lors Du Dernier Stash git stash show Permet de voir la liste des fichiers modifiés du dernier stash.
Appliquer Le Dernier Stash du Cache Git git stash apply Applique le tout dernier stash dans la pile des stash du cache de git.
Appliquer Un Stash du Cache Git git stash apply stash@{n} Applique le stash numéro n dans la pile des stash du cache de git.
Supprimer Le Dernier Stash Dans le Cache git stash pop Supprime le tout dernier stash dans la pile des stash du cache de git.
Supprimer Un Stash dans le Cache de Git git stash pop stash@{n} Supprime le stash numéro n dans la pile des stash du cache de git.
Cloner Un Repo git clone urlRepoDistant Clone le projet du repo distant vers un repo local (répertoire courant).
Cloner Un Repo git clone urlDistant pathRepoLocal Clone le projet du repo distant en précisant le répertoire pathRepoLocal qui constituera le repo local.
Supprimer TOUTES Les Branches Mergés git remote prune origin Supprime toutes les branches mergées et inutiles sur le repo distant.
Supprimer TOUTES Les Branches Mergés (Autre Methode) git fetch origin --prune Supprime toutes les branches mergées et inutiles sur le repo distant.
Syncroniser Repo Local a Partir d'Un Repo Distant git remote update origin --prune Synchronise le repo local à partir du repo distant.
Supprimer Une Branche Sur Un Repo Distant git push origin –-delete remBranch Supprime la branche de nom remBranch du repo distant.
Envoyer Une Branche dans Un Repo Avec Un Nouveau Nom git push -u origin newBranch Pousse la branche locale courante sur le repo distant au nom de newBranch.
Envoyer Une Branche dans Un Repo Avec Un Nouveau Nom (Autre Methode) git push --set-upstream origin newBranch Pousse la branche locale courante sur le repo distant au nom de newBranch.
Envoyer Les Commits Sur Un Repo Distant git push Pousse les commits de la branche locale courante vers celle distante traquée.
Fetch git fetch Actualise et vérifie si la branche locale courante est iso ou non à celle du repo distant.
Pool git pull Réalise la commande 33 et en même temps récupère en local, les éventuels commits présents sur celle distante trackée.
Changer de Branche git checkout locBranch Changement de branche : passage de la branche courante à la branche locBranch.
Copie Une Branche Distante En Local git checkout -b locBranch origin/rembranch Créer à partir de la branche distante rembranch, la branche locale locBranch et switcher vers cette dernière.
Copie Une Branche Existante En Local En Concervant Le Nom git checkout --track origin/rembranch Similaire à la commande numéro 36. Ici locBranch a le même nom que rembranch.
Commit Sur La Branche Courante git cherry-pick commit-sha1 Commit sur la branche courante, un commit identifié par commit-sha1 effectué ailleurs.
Merger Une Branche git merge locbranch Merge la branche de nom locbranch dans la branche courante (pas de perte d'historique).
Restaurer L'Historique Git git rebase locbranch Rebase la branche de nom locbranch dans la branche courante (perte d'historique).
Annuler le Dernier Commit git reset --hard HEAD^ Force l'annulation du dernier commit avec suppression de l'historique.
Creer Un Commit Inversé git revert HEAD^ .. HEAD Crée un commit inverse annulant le dernier commit pushé sur le repo distant.
Creer Un Commit Inversé git revert commit-sha1 Crée le commit inverse annulant un commit précisé par son commit-sha1.
Configuration de Git git config --global clef val Affecte la valeur val au champ clef de la config global git.
Configuration de Git git config --global --unset clef Supprime la clef de la configuration globale de git.
Montrer La Configuration de Git git config --list Liste tous les paramètres (clef = valeur) de la config global git.
Annuler git checkout -- pathToFile Annule les modifications en cours effectuées sur le fichier pathToFile.
Deplacer Fichier git rm --cahed pathToFile Deplace le fichier pathToFile de la zone d'index vers le working set.
Comparer git diff pathToFile Compare les modification en cours du fichier pathToFile par rapport à sa version sur le repo local.
Ajouter git add pathToFile Ajoute le fichier modifié pathToFile dans la zone d'index pour qu'il soit prêt à être commité.
Appliquer git commit -m "commentaire" Commit dans le repo local tous les fichiers présents dans la zone d'index en ajoutant un commentaire propre obligatoire.
Creer Depot git init Marque le répertoire courant comme étant un repo local Git. Concrètement crée le dossier .git dans ce répertoire.
Repo git branch --merged | xargs git branch -d Supprime sur le repo local toutes les branches qui ont été mergée dans une autre branche.
Repo git remote add origin pathToRemoteProject.git Indique à un repo git local vierge, quel est le repo distant dont il en est l'image.
Repo git remote set-url origin pathToRemoteProject.git Change sur un repo git local non vierge, l'url du repo distant dont il en est l'image.
