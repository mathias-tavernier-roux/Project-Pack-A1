Visual Studio est L'Editeur de Code De Microsoft

Sa Version La Plus Utilisé Par Les Developpeurs Indépendants est La Version Communautaire (Communauty)

Ce Dernier Contient Non Seulement Les Outils de Developpement Web Mais Aussi Les Compilateurs Pour Les Programmes C,C++,Node JS, ...

En Plus de Garder Les Plugins / Extensions Pour Vous Simplifier La Tache en Plus Des Outils Microsoft Comme Intelicode ...

Avantages

- Supporte Une Grande Partie des Langages De Programmation (Qu'Il Soit Web Ou General)

- A Les Meme Voir Plus de Fonctions Que Visual Studio Code

Inconvenients

- Vu Que Les Langages Machine Comme C,C++ ... Require Un Compilateur Specifique

Visual Studio Est Evidement Plus Volumineux que Visual Studio Code
