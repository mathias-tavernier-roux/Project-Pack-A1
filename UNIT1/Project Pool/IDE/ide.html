Qu'est ce qu'un IDE

IDE Ou Interface de Developpement Pour Les Non Initié a La Programmation est Un Programme Conçu Pour Créer d'Autres Programmes

Il Existe Trois Type d'IDE : Web, Machine Et Les Moteur de Jeux

L'IDE Web Utilisera Et Supportera Toute Les Technologies Lié Au Developpement Web

L'IDE Machine Utilisera Et Supportera Toute Les Technologies Hors Developpement Web (C++, C# ...)

Les Moteurs De Jeux Sont Des IDE Spécialisé Pour le Developpement de Jeux Vidéo
